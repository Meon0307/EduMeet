EDUMEET HOMEPAGE

Files:
- index.html
- style.css
- script.js
- assets/

How to run:
1. Open this folder in Visual Studio Code.
2. Install the "Live Server" extension.
3. Right-click index.html.
4. Select "Open with Live Server".

Important:
- Replace assets/lecturer-photo.png with the lecturer's original high-resolution photo.
- Update the email address, telephone number, page links and button links.
- The current visual assets were cropped from the supplied reference image, so use original images for a sharper final website.
